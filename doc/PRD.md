# MermaidFlow - 产品需求文档

## 1. 产品概述

MermaidFlow 是一款在线 Mermaid 图表编辑器，帮助用户快速创建、编辑和渲染架构图与流程图。用户可以通过代码输入或 AI 智能生成两种方式创建图表，并支持 AI 语法修复功能。高级 AI 功能需要用户登录订阅后使用。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 使用基础编辑器功能 |
| 注册用户 | 邮箱/手机号注册 | 使用基础功能 |
| 订阅用户 | 付费订阅 | 使用全部 AI 功能（智能生成、语法修复） |

### 2.2 功能模块

1. **首页/编辑器页面**：主编辑器界面，包含代码编辑区、实时预览区、AI 功能面板
2. **AI 生成面板**：通过自然语言描述生成 Mermaid 图表
3. **语法修复面板**：AI 自动修复 Mermaid 语法错误
4. **用户认证系统**：登录、注册、订阅管理
5. **订阅管理页面**：查看订阅状态、升级/降级套餐

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 编辑器页面 | 代码编辑器 | Monaco Editor，支持语法高亮、行号显示、错误提示 |
| 编辑器页面 | 实时预览区 | Mermaid 实时渲染，支持缩放、导出 PNG/SVG |
| 编辑器页面 | AI 智能生成面板 | 输入自然语言描述，AI 生成 Mermaid 代码 |
| 编辑器页面 | AI 语法修复面板 | 检测语法错误，一键 AI 修复 |
| 认证页面 | 登录/注册表单 | 邮箱注册、登录、密码重置 |
| 订阅页面 | 订阅管理 | 显示当前套餐、升级选项、支付入口 |

## 3. 核心流程

### 3.1 基础编辑流程

```
用户打开页面 → 输入/粘贴 Mermaid 代码 → 实时预览渲染 → 如有错误显示错误信息 → 用户修改代码 → 循环直到成功
```

### 3.2 AI 生成流程

```
用户登录验证 → 输入自然语言描述 → 调用 AI API 生成 Mermaid 代码 → 代码填入编辑器 → 实时预览
```

### 3.3 AI 修复流程

```
用户输入代码 → 检测到语法错误 → 点击"AI 修复" → 调用 AI API 修复代码 → 修复后代码填入编辑器 → 实时预览
```

### 3.4 订阅流程

```
用户点击高级功能 → 检测未登录 → 跳转登录/注册 → 登录后检测未订阅 → 跳转订阅页面 → 完成支付 → 解锁 AI 功能
```

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深邃极简主义 + 科技感光效
- **主色调**：深色背景 `#0D1117`，强调色 `#58A6FF`（科技蓝）
- **辅助色**：成功绿 `#3FB950`，警告橙 `#D29922`，错误红 `#F85149`
- **字体**：
  - 标题：`JetBrains Mono`（等宽科技感）
  - 正文：`Inter`（清晰可读）
- **布局**：左右分栏（编辑区 | 预览区），顶部工具栏
- **图标**：Lucide Icons，线性风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 编辑器页面 | 顶部工具栏 | Logo、文件操作按钮、AI 功能入口、用户头像/登录 |
| 编辑器页面 | 左侧编辑区 | Monaco Editor、语法高亮、错误行标记 |
| 编辑器页面 | 右侧预览区 | Mermaid 图表渲染、缩放控制、导出按钮 |
| 编辑器页面 | 底部状态栏 | 错误数量、字符统计、同步状态 |
| 编辑器页面 | AI 面板 | 侧滑抽屉，聊天风格输入框 |
| 认证页面 | 登录/注册卡片 | 居中卡片，表单元素，品牌元素 |
| 订阅页面 | 套餐卡片 | 价格、功能列表、CTA 按钮 |

### 4.3 响应式策略

- **桌面端**：左右分栏布局，完整功能
- **平板端**：可切换编辑/预览模式
- **移动端**：单栏布局，底部 Tab 切换

### 4.4 交互动效

- 页面切换：淡入淡出 300ms
- 面板展开：滑入 + 轻微弹性
- 按钮悬停：微光效果 + 轻微上移
- 预览更新：0.3s 防抖延迟

## 5. 验收标准

### 5.1 功能验收

- [ ] Mermaid 代码编辑器支持语法高亮
- [ ] 实时预览区正确渲染 Mermaid 图表
- [ ] 代码错误时显示明确的错误信息
- [ ] AI 生成功能仅对订阅用户开放
- [ ] AI 修复功能仅对订阅用户开放
- [ ] 导出功能支持 PNG 和 SVG 格式
- [ ] 用户可以注册、登录、管理订阅

### 5.2 视觉验收

- [ ] 深色主题，符合科技极简风格
- [ ] 字体清晰，代码可读性良好
- [ ] 动画流畅，无明显卡顿
- [ ] 移动端布局适配正确
