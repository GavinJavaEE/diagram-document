# DiagramAI

基于 Mermaid 的在线图表编辑器与 Markdown 文档工具。用代码画图表，用文档写想法，数据存本地，隐私有保障。

## 核心功能

### 1. Mermaid 图表编辑

<img width="1920" height="963" alt="image" src="https://github.com/user-attachments/assets/1da2504f-8afd-4954-ae52-012963c78fea" />


- 代码驱动绘图，实时预览渲染
- 语法高亮与自动格式化
- 支持 PNG、SVG、PDF 导出

### 2. Markdown 文档

<img width="1920" height="963" alt="image" src="https://github.com/user-attachments/assets/d36daa17-b024-40e3-821d-191b068a9b9b" />

- 所见即所得的文档编辑器
- 内嵌 Mermaid 代码块实时渲染
- 图文混排，文档与图表一体化

### 3. AI 辅助创作

- 自然语言描述生成图表
- 对话式修改调整
- 语法错误自动提示与修复

### 4. 分享与导出

- 开启公开访问生成只读链接
- 支持导出 PDF、Markdown 源文件

## 双模式存储

### 本地模式（免登录）

<img width="1920" height="963" alt="image" src="https://github.com/user-attachments/assets/a879a48b-de21-4c1a-91cb-790fbe4ff58c" />


- 打开网站即可使用，无需注册账号
- 数据存储在浏览器本地，不上传服务器
- 断网也能正常编辑和查看
- 自动保存，关闭浏览器不丢失
- 本地最多 50 篇图表 + 50 篇文档

### 云端模式（登录后）

- 多设备同步，换电脑也能继续编辑
- AI 辅助生成与修改图表（对话式）
- 一键生成公开链接，分享给他人
- 云端存储空间更大（不受本地限额）
- GitHub 快捷登录，注册只需几秒

## 支持的图表类型

- **流程图 (Flowchart)**：业务流程、决策树
- **时序图 (Sequence Diagram)**：系统交互、消息传递
- **类图 (Class Diagram)**：面向对象设计
- **饼图 (Pie Chart)**：数据可视化
- **甘特图 (Gantt)**：项目时间线
- **状态图 (State Diagram)**：状态转换

## 工作流程

1. **编写代码**：用 Mermaid 语法编写图表，或在 Markdown 文档中混排文字与图表，编辑器实时预览
2. **AI 辅助**：用自然语言描述需求，AI 帮你生成或修改图表，语法错误自动提示
3. **分享导出**：一键开启公开链接分享给他人，或导出为 PDF、PNG、SVG 等格式归档

## 特色优势

- 无需注册即可使用，零门槛上手
- 数据默认存储在本地浏览器，隐私安全有保障
- 登录后无缝升级到云端，不丢失已有数据
- 支持深色模式，保护视力
- 响应式设计，适配桌面与移动端
